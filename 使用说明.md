# Claude 配置切换器 v4 — 使用说明

## 第一次使用

1. 把整个文件夹放到你喜欢的位置（比如桌面或文稿）
2. **右键点击「启动.command」→ 选择「打开」**
   （第一次必须右键打开，之后双击就行）
3. 弹出安全提示时点「打开」
4. 浏览器会自动打开工具页面 ✅

## 关于写入 ~/.zshrc 的逻辑

- 点「同步到 .zshrc」后，工具会**删除旧的 ANTHROPIC_AUTH_TOKEN / ANTHROPIC_BASE_URL 行**，然后追加新的
- **不是注释，是彻底替换** — 这样不会有多行冲突
- 同步完毕后记得在终端执行：`source ~/.zshrc`
- 如果你手动改过 ~/.zshrc，工具不会破坏其他内容，只影响 ANTHROPIC_ 开头的两行

## 导入 / 导出备份

1. 点顶部「📦 导入导出」Tab
2. **导出**：点「一键导出 .json」，会下载一个带日期的备份文件
3. **导入**：点「选择文件」选备份文件，或直接粘贴 JSON 到文本框
4. 选择导入模式：
   - **合并** — 保留现有配置，追加备份中没有的
   - **替换** — 清空现有，完全用备份覆盖
5. 点「确认导入」

## 历史快照

每次点「同步到 .zshrc」前，工具会自动保存一条快照，最多保留 10 条。
在「📦 导入导出」Tab 最底部可以查看并一键还原。

## 常见问题

**提示「无法打开，因为无法验证开发者」**
→ 右键点击文件 → 选择「打开」→ 点「打开」

**提示「缺少 Node.js」**
→ 前往 https://nodejs.org，下载 LTS 版本安装，装完再双击启动

**为什么手动打开 ~/.zshrc 看不到变化？**
→ 检查：① 是否真的点了「同步到 .zshrc」按钮；② 用 `grep ANTHROPIC ~/.zshrc` 验证

**使用完要关闭吗？**
→ 不关也没事。要关请在终端运行：`lsof -ti:7823 | xargs kill`
